# La recherche des galaxies popIII

Le fichier `redshift` contient le code permettant de calculer le redshift de l'ensemble des objets d'un dossier. Selon les spectres et leur origine, certaines informations peuvent changer. Ainsi, le nombre de variable des données du fichier va changer. Il faut modifier les lignes 43, 45, 153, 155, 263 et 265 en concervant autant de lettres qu'il y a de varibles dans le fichier fits observé.
